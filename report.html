Report: Future Me - 2034 Website

Introduction

This website is designed to reflect my professional journey by 2034. It showcases my work, skills, and future growth areas, targeting potential employers and collaborators.

Inspiration

I drew inspiration from three professional portfolio sites:

  • Site 1: Provided insight into minimal design.
  • Site 2: Showcased how to use media effectively.
  • Site 3: Offered ideas for accessibility features.

Accessibility

  • Alt text for images.
  • Keyboard navigability.
  • High contrast for readability.

Usability

  • Simple navigation.
  • Clear and consistent layout.
  • Responsive design.

Learning

  • Learning responsive design principles.
  • Understanding accessibility guidelines.
  • Using JavaScript for form validation.

Evaluation

The design was successful in creating a clean, professional portfolio site. Future improvements could involve more advanced animations and interactions.
